# mp言語 compiler
mp言語用のコンパイラーです  

## 概要
ダウンロードされる実行ファイルの大きさは100MBに満たないくらいのサイズです  
対応するのはWindowsのみです  
それぞれ32bit、64bit、ARM64版が存在します  
原則、Windows10以降での使用を想定しています  

## 注意
コードファイルのエンコードはUTF-8として受け入れられます（ASCII文字だけなら気にする必要はありません）
最適化はデフォルトでオンになっています

## 使い方
mpcom.exeをコーンソール上で起動します  
その際の引数は以下です  
・｛コードファイルのパス｝ ｛中間出力ディレクトリ｝ ｛出力ファイルのパス｝  
・｛コードファイルのパス｝ ｛出力ファイルのパス｝  
   //この場合は出力ファイルが予定されるディレクトリに中間出力（一時ファイル）が生成されます  
・ -no｛コードファイルのパス｝ ｛中間出力ディレクトリ｝ ｛出力ファイルのパス｝  
   //-noを第一引数に入れると最適化が無効化されます
・ -no｛コードファイルのパス｝ ｛出力ファイルのパス｝  
・version  
   //内部で使用しているシステムのバージョンの関係で意図されるバージョンではない場合があります
・help

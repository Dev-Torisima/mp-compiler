# -p compiler
-p（mp言語）用のコンパイラーです

## 概要
ダウンロードされる実行ファイルの大きさは100MBに満たないくらいのサイズです
対応するのはWindowsのみです
それぞれ32bit、64bit、ARM64版が存在します

## 注意
v1.4以前は使用しないでください（重大な問題があるため）

## 使い方
mpcom.exeをコーンソール上で起動します
その際の引数は以下です
・｛コードファイルのパス｝ ｛中間出力ディレクトリ｝ ｛出力ファイルのパス｝
・｛コードファイルのパス｝ ｛出力ファイルのパス｝ //この場合は出力ファイルが予定されるディレクトリに中間出力（一時ファイル）が生成されます
・version
・help
